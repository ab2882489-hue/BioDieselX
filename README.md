 🚛 BioDieselX

### Turn Waste into Low-Cost Diesel for Transport using AI

---

## 📌 Overview

**BioDieselX** is a smart web-based platform that simulates the production and management of **low-cost biodiesel** generated from waste materials such as used cooking oil, animal fats, and agricultural residue.

The platform is designed to address the **rising fuel costs** by providing an **affordable, sustainable, and scalable diesel alternative** for cars, buses, trucks, and public transport systems.

---

## 🚀 Features

### ⚙️ Fuel Production Calculator

* Converts waste oil into biodiesel
* Real-time output simulation

### 💰 Cost Comparison System

* Compare diesel vs biodiesel prices
* Shows percentage savings

### 🧠 AI Prediction Module (Simulated)

* Predicts fuel yield
* Recommends optimal blend (B20 / B50 / B100)

### 📊 Data Visualization

* Fuel production trends
* Cost savings analytics using charts

### 🗺️ Fuel Availability System

* Displays nearby biodiesel stations (simulated)
* Shows pricing and availability

### ♻️ Waste Collection Module

* Register waste suppliers (restaurants, businesses)
* Input waste quantity for fuel production

### 🌱 Environmental Impact Tracker

* CO₂ emission reduction stats
* Waste recycled metrics

## 🔬 Core Concept

BioDieselX is based on the **transesterification process**, where waste oil is converted into biodiesel using a chemical reaction involving alcohol and a catalyst.

This allows biodiesel to be used in **existing diesel engines** with little or no modification.
